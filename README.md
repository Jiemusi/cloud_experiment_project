# cloud_experiment_project

What to do on a new VM:

1. Start and test run:
```
chmod +x start.sh test_run.sh
./start.sh
./test_run.sh

Maybe install ncu here?

```
2. Start running runs without ncu:
```
chmod +x run_without_ncu.sh
run_without_ncu.sh
```

3. Run with ncu:
```
chmod +x run_with_ncu.sh
run_with_ncu.sh
```
